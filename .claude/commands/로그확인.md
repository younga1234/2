# 로그 확인

## 1. 오늘의 세션 로그 확인

```bash
# 오늘 날짜의 로그 파일 읽기
cat .claude/logs/session_$(date +%Y-%m-%d).log
```

## 2. 로그 파일 목록

```bash
# 모든 로그 파일 목록
ls -lah .claude/logs/*.log

# 아카이브된 로그 (10MB 초과 시 자동 아카이브)
ls -lah .claude/logs/archive/*.log 2>/dev/null || echo "아카이브된 로그 없음"
```

## 3. 로그 분석

다음 정보를 로그에서 추출하여 정리하세요:

### 📊 세션 통계
```bash
# 세션 시작/종료 횟수
grep "SESSION_START\|SESSION_END" .claude/logs/session_$(date +%Y-%m-%d).log | wc -l

# 실행된 슬래시 커맨드 목록
grep "COMMAND" .claude/logs/session_$(date +%Y-%m-%d).log
```

### 🤖 에이전트 사용 내역
```bash
# 실행된 에이전트 목록
grep "AGENT" .claude/logs/session_$(date +%Y-%m-%d).log
```

### 💾 Git 커밋 기록
```bash
# 로그에 기록된 커밋들
grep "GIT_COMMIT" .claude/logs/session_$(date +%Y-%m-%d).log
```

### 🗜️ 컨텍스트 압축 기록
```bash
# Compact 실행 기록
grep "COMPACT" .claude/logs/session_$(date +%Y-%m-%d).log
```

## 4. 특정 기간 로그 검색

```bash
# 최근 7일간의 로그
find .claude/logs -name "session_*.log" -mtime -7 -exec cat {} \;

# 특정 날짜 범위
# 예: 2025-12-01부터 2025-12-05까지
for date in {01..05}; do
    cat .claude/logs/session_2025-12-$date.log 2>/dev/null
done
```

## 5. 로그 검색 패턴

### 오류 검색
```bash
grep -i "error\|fail\|warning" .claude/logs/session_$(date +%Y-%m-%d).log
```

### 특정 작업 검색
```bash
# 예: "authentication" 관련 작업
grep -i "authentication\|auth" .claude/logs/session_$(date +%Y-%m-%d).log
```

### 시간대별 활동
```bash
# 특정 시간대 (예: 09시-12시)
grep "09:\|10:\|11:" .claude/logs/session_$(date +%Y-%m-%d).log
```

## 6. 로그 요약 보고서

다음 형식으로 오늘의 작업을 요약하세요:

### 📅 작업 일자
- **날짜**: $(date +%Y-%m-%d)
- **세션 수**: X회

### ⏱️ 활동 시간
- **첫 세션 시작**: HH:MM
- **마지막 세션 종료**: HH:MM
- **총 작업 시간**: X시간

### 🎯 주요 작업
로그에서 추출한 주요 활동:
- 실행된 커맨드: /현황파악, /테스트디버깅, ...
- 사용된 에이전트: code-reviewer, ...
- Git 커밋: X개

### 📝 작업 내용
로그를 분석하여 오늘의 주요 작업 내용 요약:
- [작업 1] - 시간대
- [작업 2] - 시간대
- [작업 3] - 시간대

### 📊 통계
- 커맨드 실행 횟수: X회
- 에이전트 호출 횟수: X회
- Git 커밋 횟수: X회
- 컨텍스트 압축 횟수: X회

## 7. 로그 관리

### 로그 정리
```bash
# 30일 이상 된 로그 아카이브로 이동
find .claude/logs -name "session_*.log" -mtime +30 -exec mv {} .claude/logs/archive/ \;
```

### 로그 백업
```bash
# 로그 전체 백업
tar -czf logs_backup_$(date +%Y%m%d).tar.gz .claude/logs/
```

### 디스크 사용량 확인
```bash
# 로그 디렉토리 크기
du -sh .claude/logs
du -sh .claude/logs/archive
```

## 8. 로그 설정

### 자동 로깅 활성화 상태
```bash
# 로거 스크립트 존재 여부
if [ -f "hooks/auto-logger.sh" ]; then
    echo "✅ 자동 로깅 활성화됨"
    chmod +x hooks/auto-logger.sh
else
    echo "❌ 자동 로깅 비활성화됨"
fi
```

### 로그 레벨 (현재 기록되는 항목)
- SESSION_START: 세션 시작
- SESSION_END: 세션 종료
- COMMAND: 슬래시 커맨드 실행
- AGENT: 에이전트 호출
- GIT_COMMIT: Git 커밋
- COMPACT: 컨텍스트 압축
- INFO: 일반 정보
- ARCHIVE: 로그 아카이브
