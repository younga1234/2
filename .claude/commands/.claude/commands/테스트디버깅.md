# 테스트 디버깅

## 1. 전체 테스트 실행

```bash
python -m pytest tests/ -v --tb=short
```

## 2. 실패한 테스트 상세 분석

각 실패한 테스트에 대해:

1. **테스트 파일 읽기**: 실패한 테스트 코드 확인
2. **관련 소스 코드 읽기**: 테스트 대상 함수/클래스 분석
3. **실패 원인 파악**:
   - 예상 값과 실제 값 비교
   - 논리 오류 확인
   - 환경 설정 문제 확인
   - 의존성 버전 호환성 확인

## 3. 디버깅 결과 보고

다음 형식으로 정리하세요:

### 🔴 실패한 테스트 목록
- `test_function_name`: 간단한 설명

### 🔍 원인 분석
각 테스트별:
- **위치**: `tests/test_file.py::test_name`
- **원인**: 구체적인 문제점
- **심각도**: Critical / High / Medium / Low

### 🛠️ 수정 방안
- 우선순위별 수정 제안
- 예상 코드 변경 사항

### ✅ 수정 후 재실행
문제 해결 후 다시 테스트 실행하여 검증
