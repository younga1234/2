# 🎓 초보자 모드 (Super Easy Mode)

비개발자도 20년차 시니어 개발자처럼 개발할 수 있게 도와드립니다!

## 🚀 시작하기

### 1단계: 무엇을 만들고 싶은지 말해주세요

사용자에게 질문하세요:
```
💬 질문: "어떤 프로그램을 만들고 싶으세요?"

예시 답변:
- "고객 관리 프로그램"
- "날씨 정보 보여주는 웹사이트"
- "재고 관리 시스템"
- "챗봇"
- "데이터 분석 대시보드"
```

### 2단계: 자동으로 모든 것 설정

사용자 답변을 받으면 다음을 **자동으로** 수행:

```bash
# 시니어 멘토 에이전트 호출
/agent senior-mentor "사용자가 원하는 것 분석하고 쉬운 말로 설명"

# 자동 아키텍트 에이전트 - 모든 기술적 결정 자동 처리
/agent auto-architect "최적의 기술 스택 선택 및 프로젝트 구조 생성"

# 프로젝트 마법사 - 전체 프로젝트 자동 생성
/agent project-wizard "즉시 실행 가능한 프로젝트 생성"
```

### 3단계: 사용자에게 결과 알림 (쉬운 말로)

```markdown
✅ 완료! 당신의 프로그램이 준비되었습니다!

📁 생성된 것:
- 프로젝트 폴더 ✓
- 필요한 모든 파일 ✓
- 데이터베이스 설정 ✓
- 바로 실행 가능한 코드 ✓

🎯 지금 할 수 있는 것:
1. 프로그램 실행하기: `python main.py` 입력
2. 웹 브라우저에서 보기: http://localhost:8000
3. 코드 수정하기: 아래 가이드 참조

📖 쉬운 설명:
[auto-architect가 선택한 기술을 쉽게 설명]
- FastAPI를 선택했어요: 빠르고 안정적인 웹 프로그램을 만들 수 있어요
- PostgreSQL을 선택했어요: 데이터를 안전하게 저장할 수 있어요
```

## 🛠️ 자동 기능

### 자동 에러 해결

에러가 발생하면 **자동으로**:

```bash
# 초보자 도우미가 자동 개입
/agent beginner-helper "에러 자동 감지 및 수정"
```

사용자에게는:
```
🔧 문제를 발견했어요!
❌ 문제: 필요한 도구가 설치되지 않았습니다
✅ 해결: 자동으로 설치했습니다 (pandas, numpy 설치 완료)

이제 다시 실행하면 잘 작동할 거예요! 😊
```

### 자동 코드 설명

사용자가 "이 코드가 뭐예요?"라고 물으면:

```bash
/agent code-explainer "일반인이 이해할 수 있게 설명"
```

결과:
```
📖 코드 설명 (쉬운 버전)

def get_users():
    return db.query(User).all()

→ 쉬운 설명:
"데이터베이스(저장소)에서 모든 사용자 정보를 가져오는 함수예요.
 마치 도서관에서 모든 책 목록을 가져오는 것과 비슷해요!"

왜 이렇게 작성했나요?
- 재사용 가능: 여러 곳에서 쓸 수 있어요
- 깔끔함: 한 번에 한 가지 일만 해요
```

### 예제 2: 에러 발생 시

**문제**: ModuleNotFoundError: No module named 'fastapi'

**자동 해결**:
```bash
# 자동으로 감지 및 수정
/agent beginner-helper "에러 자동 해결"
```

**사용자에게 표시**:
```
🔧 문제를 자동으로 해결했어요!

"FastAPI라는 도구가 없어서 프로그램을 실행할 수 없었어요"

✅ 해결 방법:
- FastAPI 자동 설치 완료
- 관련 도구들도 함께 설치 (uvicorn, pydantic)
- 설정 파일 업데이트

🎉 이제 프로그램이 잘 실행될 거예요!

앞으로 이런 문제가 다시 발생하지 않도록:
- requirements.txt 파일을 자동으로 업데이트했어요
- 다음부터는 자동으로 설치될 거예요
```

### 예제 3: 코드 이해하기

**사용자**: "이 코드가 무슨 뜻이에요?"

```python
@app.post("/items/")
async def create_item(item: Item):
    return {"item_id": item.id}
```

**자동 설명**:
```bash
/agent code-explainer "쉬운 설명 제공"
```

**결과**:
```
📖 코드 쉬운 설명

이 코드는 "새 상품 추가하기" 기능이에요!

🏪 비유:
마치 가게 주인이 새 상품을 진열대에 올리는 것과 같아요.

📝 각 부분 설명:
- @app.post: "여기에 새 것을 추가해주세요" 라는 표지판
- "/items/": 상품을 추가하는 위치 (주소)
- create_item: 이 작업의 이름 (상품 만들기)
- item: Item: 받을 정보 (어떤 상품인지)
- return: 결과 알려주기 (상품 번호)

🎯 실제로는:
1. 사용자가 "새 상품 추가" 버튼 클릭
2. 이 코드가 실행됨
3. 데이터베이스에 저장
4. "상품이 추가되었어요!" 메시지 표시

💡 왜 async를 쓰나요?
- 빠른 처리: 여러 사용자가 동시에 상품을 추가해도 괜찮아요
- (집에서 요리하면서 빨래도 동시에 하는 것처럼!)
```
