# 🎯 다음에 무엇을 할까요?

시니어 개발자처럼 최적의 다음 단계를 자동으로 제안합니다!

## 1. 현재 프로젝트 상태 자동 분석

```bash
# 자동으로 분석
git status
ls -la
cat requirements.txt 2>/dev/null || echo "No requirements.txt"
```

## 2. 시니어 멘토 에이전트 호출

```bash
/agent senior-mentor "프로젝트 상태 분석 및 다음 단계 제안"
```

## 3. 단계별 제안 (자동 우선순위)

다음 형식으로 제안하세요:

### 🚨 긴급 (지금 바로 해야 함)
```
❗ 발견된 문제:
- [ ] 테스트가 실패하고 있어요 (3개)
     → 자동으로 수정할까요? (Y/n):

- [ ] 보안 취약점 발견 (SQL Injection)
     → 지금 바로 수정합니다! (자동 실행 중...)
```

### ⭐ 중요 (오늘 안에 하면 좋음)
```
💡 권장 작업:
1. 데이터베이스 백업 설정
   → 자동으로 설정할까요? (Y/n):
   → 예상 시간: 2분
   → 효과: 데이터 손실 방지

2. API 문서 업데이트
   → 자동으로 생성할까요? (Y/n):
   → 예상 시간: 3분
   → 효과: 팀원들이 사용하기 쉬워요
```

### 📈 개선 (시간 날 때 하면 좋음)
```
🎨 더 나아지려면:
1. 코드 성능 30% 개선 가능
   → 어떻게?: 데이터베이스 인덱스 추가
   → 자동으로 할까요? (Y/n):

2. UI 디자인 개선
   → 제안: Bootstrap → Tailwind CSS
   → 나중에 알림 설정할까요? (Y/n):
```

### 🎓 학습 (새로운 것 배우기)
```
📚 배울 만한 것:
1. 비동기 프로그래밍 (async/await)
   → 왜 필요?: 더 빠른 프로그램
   → 5분 튜토리얼 보여줄까요? (Y/n):

2. Docker 컨테이너
   → 왜 필요?: 어디서나 똑같이 실행
   → 쉬운 가이드 보여줄까요? (Y/n):
```

## 4. 자동 실행 모드

사용자가 "Y"를 입력하면 자동으로 실행:

### 예제: 테스트 수정
```bash
# 사용자: Y

# 자동 실행
/agent test-engineer "실패한 테스트 자동 수정"

# 결과 표시
✅ 완료! 3개의 테스트를 수정했어요!

수정 내역:
1. test_user_login: 잘못된 비밀번호 테스트 수정
2. test_api_endpoint: URL 경로 수정
3. test_database: Mock 데이터 추가

이제 모든 테스트가 통과해요! 🎉
```

### 예제: 데이터베이스 백업 설정
```bash
# 사용자: Y

# 자동 실행
/agent devops-engineer "자동 백업 스크립트 생성"

# 결과 표시
✅ 완료! 데이터베이스 자동 백업이 설정되었어요!

📋 설정 내용:
- 백업 주기: 매일 새벽 2시
- 저장 위치: backups/ 폴더
- 보관 기간: 30일
- 자동 암호화: 활성화

다음 백업: 내일 02:00 AM

안심하세요! 데이터는 안전해요 😊
```
